# 🏥 疫苗预约与接种管理系统（全栈项目）

## 📌 项目简介
本项目为儿童疫苗预约与接种管理系统，采用前后端分离架构，实现家长端、医生端、管理员端三端业务闭环。

系统围绕“预约 → 审核 → 接种 → 记录 → 统计”完整业务流程设计。

---

## 🛠 技术架构

### 🔹 后端
- Spring Boot 3.x
- MyBatis-Plus
- MySQL 8
- Spring Security
- RESTful API
- Swagger

👉 后端仓库：
https://github.com/songzi11/vaccine-system

---

### 🔹 前端
- UniApp
- Vue3
- Vuex
- 多端支持（H5 / 小程序 / App）

👉 前端仓库：
https://github.com/songzi11/vaccine-system-frontend

---

## 🎯 核心业务模块

### 👨‍👩‍👧 家长端
- 宝宝信息管理
- 疫苗预约
- 接种记录查询
- 接种点选择

### 👨‍⚕️ 医生端
- 排班管理
- 当日预约查看
- 接种确认
- 接种记录录入

### 👨‍💼 管理员端
- 用户管理
- 疫苗管理（上下架控制）
- 库存联动管理
- 接种点管理
- 数据统计分析

---

## 📐 系统架构

前后端分离架构：

用户端 / 医生端 / 管理员端
        ↓
UniApp 前端
        ↓
RESTful API
        ↓
Spring Boot 后端
        ↓
MySQL 数据库
